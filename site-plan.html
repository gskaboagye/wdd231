Sefa Tees – Project Site Plan

Prepared by: Godfred Sefa Aboagye

Site Name

Sefa Tees

This name represents a creative T-shirt brand that focuses on modern, stylish, and custom designs. It was chosen because it directly connects with my brand and vision.

Optional domain availability: sefatees.com

Site Purpose

The purpose of this site is to promote and sell custom-designed T-shirts, showcase unique fashion ideas, feature customer reviews, and provide a contact form for bulk orders or inquiries.

Scenarios

  • How can I order a customized T-shirt with my own design?
  • Where can I see reviews from other customers before purchasing?

Color Schema

  • #1E3A8A (Dark Blue) – headings, header background, accents
  • #F59E0B (Warm Amber) – buttons, call-to-action highlights
  • #F3F4F6 (Light Gray) – background

Typography

  • Poppins – for headings
  • Roboto – for body text

This document is already styled using these fonts.
